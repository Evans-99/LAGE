This repository provides the implementation details and reproducibility materials for the LLM coordination module in LAGE.

**configs:** Configuration files for Qwen-7B inference and LLM output processing.<br>
**examples:** Representative LLM input contexts and structured output examples.<br>
**prompts:** System prompts, user prompt templates, and few-shot examples.<br>
**schemas:** JSON schemas used to validate the structured LLM responses.
